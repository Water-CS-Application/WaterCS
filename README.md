# Welcome to WaterCS

## 写在前面

本站的设计初衷是为了区分globalCS、OpenCS两站在欧陆和北美地区申请给予申请季陆本学子的指导帮助，但是globalCS对于英港新项目的具体描摹不够清晰

同时这部分市场存在大量**无良中介**，所以本站应运而生

## 如何成为贡献者
